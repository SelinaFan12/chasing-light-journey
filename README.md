# 追光旅迹

追光旅迹是一款微信小程序示例项目，面向旅行摄影和城市漫步场景，帮助用户查找拍照点位、查看现场机位建议、收藏路线线索，并快速复制导航信息。

## 功能

- 机位首页：城市、主题、关键词筛选，支持附近推荐和收藏。
- 机位详情：最佳光线、推荐机位、站位说明、拍摄动作、避坑提示和导航信息复制。
- Citywalk 路线：按城市提供起点、中段、收尾三个拍摄节点。
- 姿势库：按场景筛选旅行拍照姿势。
- 文案库：按语气匹配可复制的旅行分享短句。
- 个人页：本地收藏、浏览历史、主题切换和缓存清理。

## 技术栈

- 微信小程序原生框架
- 微信云开发云函数
- 本地静态数据兜底
- 可选接入火山方舟/豆包用于文案生成
- 可选接入授权 POI 数据用于点位同步

## 本地运行

1. 打开微信开发者工具。
2. 选择“小程序”并导入当前项目目录。
3. 如果只是预览本地示例数据，可以使用 `touristappid`。
4. 如果要部署到自己的小程序，请在 `project.config.json` 中替换为自己的 `appid`。
5. 编译后即可预览。

## 云函数和环境变量

项目默认可以只使用本地数据运行。云函数是可选增强：

- `cloudfunctions/getSpots`：从云数据库读取审核通过的点位。
- `cloudfunctions/syncXhsData`：导入授权 POI 或人工采编数据。
- `cloudfunctions/generateCaption`：调用外部模型生成文案。

需要配置的环境变量示例见 [.env.example](.env.example)。不要把真实 API Key、访问令牌或云环境密钥提交到仓库。
