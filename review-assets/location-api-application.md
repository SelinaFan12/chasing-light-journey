# 地理位置接口开通申请文案

## 建议开通接口

`wx.getFuzzyLocation`

本小程序只需要获取我的模糊地理位置，用于附近拍照机位推荐，不需要获取我的精确门牌地址、收货地址或手动选点。

## 场景描述

我在使用「追光地图」查找旅行拍照机位时，可以在首页主动点击「附近推荐」。小程序会调用 `wx.getFuzzyLocation` 获取我的大致地理位置，并根据我所在城市和附近距离，对拍照机位进行排序展示。

获取位置后，小程序只用于计算我与拍照地点之间的大致距离，例如优先展示外滩、武康路、西湖、故宫等距离更近的打卡机位，帮助我更快选择适合当前出行路线的拍照地点。

小程序不会获取我的收货地址，不会让他人查看我的位置，也不会将我的位置用于社交展示或精准追踪。若我拒绝授权，仍可以通过城市筛选和搜索功能手动查找拍照机位。

## 使用路径

首页 → 点击「附近优先」 → 弹出位置授权 → 同意后展示附近拍照机位和大致距离。

## 不建议申请的接口

- `wx.chooseAddress`：当前没有电商、收货或配送功能，不需要开通。
- `wx.chooseLocation`：当前不需要我手动在地图上选择位置，不需要开通。
- `wx.choosePoi`：当前不需要我从 POI 列表选择位置，不需要开通。
